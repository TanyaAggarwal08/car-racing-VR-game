SYNTHWAVE RACER VR

📖 GAME MANUAL & GUIDE
◈ INSTALLATION & SETUP
  1. Download or clone the project folder to your computer.
  2. Open index.html in any modern web browser (Chrome, Firefox, Edge, Safari).
    No server required — runs directly from the file system.
  3. Wait for the splash screen countdown, then click START RACE.
  4. Configure your options on the Setup screen and click START RACE.
  5. For Mobile VR (Cardboard): open in Chrome on Android, tap the VR goggles button at the bottom of the game screen, then place your phone in a Google Cardboard headset.

◈ VR & TECHNICAL NOTES
▸ Built with A-Frame 1.4 — WebXR/WebVR compatible
▸ Stereoscopic rendering activates automatically in VR mode (Google Cardboard compatible)
▸ All textures are generated procedurally at runtime — no downloads required
▸ All audio generated via Web Audio API — no external sounds
▸ Gaze cursor visible in VR as a pink reticle dot
▸ For best mobile VR: use Chrome on Android with phone in landscape inside headset
▸ Allow DeviceOrientation permission if prompted (required for head-tilt on iOS)
